# API Service

Backend for the wallet (REST/GraphQL). Recommended stack: Node.js + TypeScript (Express/NestJS), ethers.js for chain ops.

Suggested structure:
```
src/
  main.ts
  app.ts
  routes/
  controllers/
  services/
  lib/
  middlewares/
  config/
  types/

tsconfig.json
package.json
.env.example
```
